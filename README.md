# AI Personal Assistant

A simple web application with login and tools list.

## How to Run

1. cd AIPersonalAssistant
2. dotnet run --project AIPersonalAssistant.Web
3. Open browser to https://localhost:7001 or http://localhost:5000

## Features

- Login page (simple UI, no real auth yet)
- Landing page showing tools from API
- Vanilla JavaScript with modern features
- .NET 10 backend with C#10
